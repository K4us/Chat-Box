# Chat

1. Clone the repo from GitHub.

         $ git clone https://github.com/K4us/Chat-Box.git

2. Install dependencies.

         $ cd Chat-Box
         $ npm install

3. Edit sass code in  `scss/*`

4. Let run.

         $ grunt sass
         $ http-server -o
